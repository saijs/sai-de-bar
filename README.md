# sai-de-bar [![spm version](http://spmjs.io/badge/sai-de-bar)](http://spmjs.io/package/sai-de-bar)

---

Sai 的 bar.

简易控制台，开发、测试环境可以考虑启用。

---

## INSTALL

```
$ spm install sai-de-bar --save
```

## USAGE

```js
var Saidebar = require('sai-de-bar');
Saidebar.log("log");
Saidebar.info("info");
Saidebar.warn("warn");
Saidebar.error("error");
```

## API

### .log(String message)

### .info(String message)

### .warn(String message)

### .error(String message)

### .open()

### .close()
